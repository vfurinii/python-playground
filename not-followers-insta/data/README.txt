# Diretório para arquivos JSON exportados do Instagram
# Coloque aqui os arquivos:
# - followers_1.json (ou followers.json)
# - following.json

# Para exportar seus dados:
# 1. Acesse Instagram > Configurações > Segurança > Baixar dados
# 2. Escolha formato JSON
# 3. Aguarde o email do Instagram (pode levar até 48h)
# 4. Extraia os arquivos JSON e coloque nesta pasta
